# 固件文件 | Firmware Files

本目录提供 xteink 阅星瞳 X3 / X4 的官方固件，用于恢复原厂系统。

CN/EN 标注仅为默认界面语言不同，固件内容完全一致，刷入后可在设置中切换语言。

## 固件类型说明

| 后缀 | 类型 | 大小 | 用途 |
|---|---|---|---|
| `_full.bin` | 完整备份 | 16MB | 恢复整个 Flash，包含 bootloader + 分区表 + 应用 + 数据 |
| `_ota.bin` | OTA 固件 | ~6MB | 仅更新应用分区，需设备当前运行官方固件 |

## English

Official firmware files for xteink X3 / X4 e-readers.

CN/EN designation only indicates the default UI language. Firmware content is identical; language can be switched in settings after flashing.

### File types

| Suffix | Type | Size | Usage |
|---|---|---|---|
| `_full.bin` | Full backup | 16MB | Restore entire flash (bootloader + partitions + app + data) |
| `_ota.bin` | OTA firmware | ~6MB | Update app partition only (requires official firmware) |
