# 固件文件 | Firmware Files

本目录提供 xteink 阅星瞳 X3 / X4 的官方固件备份，用于恢复原厂系统。

CN/EN 标注仅为默认界面语言不同，固件内容完全一致，刷入后可在设置中切换语言。

## X3 固件（完整备份）

完整 16MB Flash 镜像，包含 bootloader + 分区表 + 应用 + 数据。使用「恢复原厂」模式写入。

| 文件 | 语言 | 版本 |
|---|---|---|
| x3_cn_v5.2.13_full.bin | 中文 | V5.2.13 |
| x3_en_v5.2.13_full.bin | 英文 | V5.2.13 |
| x3_en_v1.0.7_full.bin | 英文 | V1.0.7 |

## X4 固件（OTA）

仅包含应用分区（~6MB），用于 OTA 更新。使用「OTA 更新」模式写入。

| 文件 | 语言 | 版本 |
|---|---|---|
| x4_cn_v5.2.13_ota.bin | 中文 | V5.2.13 |
| x4_en_v5.1.6_ota.bin | 英文 | V5.1.6 |

## English

Official firmware files for xteink X3 / X4 e-readers.

CN/EN designation only indicates the default UI language. Firmware content is identical; language can be switched in settings after flashing.

### X3 Firmware (Full Backup)

Full 16MB flash image including bootloader + partition table + app + data. Use "Restore" mode.

### X4 Firmware (OTA)

App partition only (~6MB). Use "OTA Update" mode.
